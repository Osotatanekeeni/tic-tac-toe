# Tic Tac Toe
![CircleCI](https://img.shields.io/circleci/build/github/Osotatanekeeni/tic-tac-toe)

This is a simple 3x3 tic-tac-toe game

## Running The App Locally
- Clone the repo
- Run ***npm install***
- Run ***npm start***
- View the app on your browser at ***localhost:3000***
